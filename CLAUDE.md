# CLAUDE.md — 红楼幻梦

## 项目一句话
一部 **2 小时、纯点击的"互动红楼电影"**（视觉小说 VN）：现代人穿越进元春省亲夜，用 MBTI 读人之术查一桩"通灵宝玉失踪案"，每读懂一个人就窥见 TA 的判词命运，最后被逼做一个唏嘘的道德抉择。技术栈：React 19 + TypeScript + Vite + Tailwind v4 + PWA。

> **重大转向（2026-06-18）**：本项目已从"30天养成+好感度"**推翻重做**为上述探案 VN。养成引擎弃用，美术/场景/立绘/AVG 演出/存档框架保留复用。

## 北极星（不可动摇）
**悬疑不是根本，"读懂人 + 红楼文化交融"才是根，破案只是载体。** 玩家通关记住的应是"原来她是这样一个人"，不是"原来凶手是谁"。

## 核心设计公式
- **读懂一个人 = 窥见 TA 的判词命运**。每个角色 **MBTI型 ⟷ 判词 ⟷ 案件动机** 三位一体。
- **签名机制"反过来读你"**：全程暗记玩家关键选择，结局反向照回——"你说你是X，可面对凤姐的窟窿、王夫人那只手、贾母那只闭上的眼，你究竟是谁？"
- **玩家 MBTI 受控**：不分叉16条剧情树。只作用三层——①开场旁白 ②感知透镜（N看潜台词/F看情绪/T看逻辑漏洞/J给线索板，改变"感知什么"不改剧情树）③结局回响。

## 案件宪法（每环有原著背书，勿改）
- 表层：省亲夜通灵宝玉失踪，满府翻天。
- 真相一层：**玉是宝玉自己藏的**（厌恶象征家族期望与既定姻缘的玉，原著摔玉前科；省亲夜亲手扔"命"）。玉始终安全。
- 真相二层：**凤姐**借混乱干财务勾当（挪公中/放贷/以假换真——"机关算尽"）。她不碰玉，动机是钱。
- 真相三层：小丫鬟撞破凤姐 → **王夫人**亲手冷酷清除（签名罪行：金钏投井、晴雯被撵致死，自认行善）→ **贾母**顶端默许（"别让娘娘知道"，不下令只选择不看，保元春/体面）。
- 锥心反讽：满府找玉，没人找那个陪葬的女孩，除了你。结局抉择：说出真相会炸毁家族、可能害了元春——点不点燃。
- 基调：**唏嘘而非痛快**（阿加莎式悲悯，非利刃出鞘式解气）。系列化野心：每部横切红楼一个时期。

### 立绘配方（锁定 2026-06-18，所有角色照此防漂）
- **核心发现：之前"虚"的病根不是画风，是"白底立绘抠图拼贴"→人物飘。解法＝人景一体。** 同一种水墨，人长进空间里就"实"，浮在白底就"虚"。
- **风格 = 人景一体水墨淡彩**：水墨淡彩为底 + 柔和体积光 + **落地投影**（人物脚下/窗棂影投在地面）+ 与场景共享同一光线和纵深，人景浑然一体，非抠图拼贴。
- **脸**：清雅文气内敛的红楼气质，**拒绝 90 年代男性向美少女脸/香艳取景**（受众是 MBTI 女性）。
- **要偷老智冠的"构图落地感（实）"，不偷它的"美少女脸+香艳（男性向）"——两者可分离。**
- 提示词母版（ChatGPT images）：`竖版画面，{角色}{动作}，站在{场景}里。水墨淡彩为底但有柔和体积光，人物脚下有落地投影，与{环境元素}共享同一光线和空间纵深，人景浑然一体（非抠图拼贴），清雅文气面容，电影级氛围。`
- 用法分层：**大场面/定场用"人景一体"整幅关键帧**；日常对话若需近景表情，另生**脸部特写变体**（同脸靠图片参考保一致）。
- 收尾：裁掉生成器水印；诗册等道具上的 AI 乱码无伤大雅或修掉。

## 关键业务规则
- **失败的代价＝信任/距离槽（机制地基，定于 2026-06-18）**：每个角色一条信任槽（=你和她的距离本身，非游戏化血条）。读对→走近（槽+），读错→推远（槽-），到阈值→她卸防吐真话（线索+判词裂缝）。
  - 可挽回（读错伤但能补），渐变非二元 → 满足"谁都能通关"（推远了照样有结局，只是"终究没真正懂她"版）。
  - 每个角色"宽容度"不同＝编码其 MBTI（凤姐 ESTJ 难撬难回；湘云 ENFP 好亲近）。
  - 表现为**立绘神情/姿态/话的温度的暗示**，**不画直白进度条**。
- 读人判定：玩家读法（共情/论理/奉承/硬碰/旁观…）匹配对方"钥匙"→ 读对（吐真话）；不匹配 → 读错（起表演，**仍有收获，永不死胡同**，但槽-）。
- 镜子：只追踪少数有分量的选择（用什么读法读谁、关系的形状、点不点燃），结局具体照回，不可随机贴标签。
- 线性三天推进，非自由探索；6-8 场读人相遇硬封顶。
