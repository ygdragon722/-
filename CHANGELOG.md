## [0.1.0] - 2026-05-31

### 初始版本
- 🎮 核心玩法：30 天 × 3 时段文字养成
- 👩 7 位女主角（黛玉/宝钗/湘云/探春/袭人/晴雯/妙玉），每人 3 段 MBTI 剧情
- 🗺️ 6 个地点：潇湘馆、蘅芜苑、沁芳亭、秋爽斋、怡红院、栊翠庵
- 📦 7 件道具（6 礼物 + 百年人参）+ 宁荣街商店
- 🎲 随机事件：贾政考较、贾母关爱、赵姨娘找茬
- ⚡ 固定大事件：第 15 天海棠诗社、第 25 天抄检大观园
- 🌤️ 天气系统（晴/多云/雨），影响行动收益
- 🏆 9 种结局 + 太虚幻境图鉴收集
- 💾 localStorage 自动存档/续玩

### 技术栈
- React 19 + TypeScript + Vite
- Tailwind CSS v4
- 自定义游戏引擎（useReducer + useCallback）
