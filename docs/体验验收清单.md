# 体验验收清单 · 通关审稿 · 2026-06-20

> 方法：通读全部叙事数据文件（opening/day1三场/day2两场/day3两个抉择/Day1Wrap/Day2Wrap/Ending）+ 在预览里实际走一遍抉择场的点击流程。不改代码，只给判断。
> 结论先行：主链条本身是成立的，**不需要再加人物/支线/系统**。真正要做的是节点级的"力度"补强——大多数问题不是"缺什么"，是"有，但还不够使劲"。

## 1. 玩家是否明白自己是谁、为什么入梦

**基本成立，但有一处隐性跳跃没挑明。**

> 2026-06-21 已修补：开场第一拍补出现代仿建大观园景区、检票闸机、玻璃门中映出的年轻女人和导览图；后续已补“你现在，就在贾宝玉的身体里”的内心确认。

- 开场前两拍（"你读了十年红楼""你尤其懂宝玉"）把玩家的自我认知立住了：一个自信"比角色更懂角色"的现代读者。这个起点很好，后面"反过来读你"的落差全靠它撑着。
- 道人+镜子三连拍（现在已经有画面了，效果好）交代了"附身"的机制，但**没有一句话明确说"你现在在宝玉的身体里"**——是靠"镜子里是他的脸"+丫鬟喊"宝二爷"两个间接信号拼出来的。对照过原著、反应快的玩家能秒懂；但这是"互动短篇"，**不能要求每个玩家都做对照阅读**。
- **建议**：冷开场最后一拍或刚入梦那一刻，加半句**玩家内心的自认**（哪怕一句"——这身体，是宝玉的"），把推理性的"拼出来"变成确定性的"被告知"。这不是加内容，是把已有的暗示挑明一句话的事。

## 2. 每一天信息是否逐步升级，而不是"又见一个人"

**这条做得相当好，三天的"性质"是真的不同，不是同一个机制换皮重复。**

- **第一天**：凤姐/黛玉/王夫人三场，揭的是"人"——每个人要的读法完全不同（凤姐要被看见、黛玉要不被催、王夫人要顺服），这一天结束揭示真相第一层（玉是宝玉自己藏的）。性质：**解谜，轻**。
- **第二天**：贾母+王夫人佛堂，揭的是"案"——从"性格画像"跳到"道德恐怖"，贾母那句"查得太清楚，谁都没好处"、王夫人那句"慈悲，有时候是要沾手的"，两句话的杀伤力明显比第一天任何一句台词都重。Day2Wrap 收尾那句"一块玉，惊动了整座府第……一个活生生的女孩子没了，月例册子上轻轻一划，就像她从没来过"，是全剧目前写得最好的一句话，直接把主题摆到桌面上。性质：**道德，重**。
- **第三天**：揭的是"你"——前两天积累的认知失重，落到玩家自己手里做决定。
- 升级曲线是对的：**人→案→你**，一天比一天往玩家身上逼近。这个结构不需要动。

**唯一的衔接小问题**：Day2Wrap 结尾用"说出来，还是，像所有人一样，把它咽下去"来预告第三天，但第三天实际给的是两个**独立**的二元抉择（玉留不留／女孩记不记得），不是一次"说真话还是噎下去"的单一动作。玩家读完 Day2Wrap 的预告后，可能在期待一个"要不要揭穿凶手"式的摊牌时刻，结果等到的是两次"处置"型选择——落差不大，但措辞上有一点点没对齐。
**建议**：把 Day2Wrap 最后这句换成更准确预告"两件事都要你来决定"的措辞，不要用"说出来/咽下去"这个后面用不上的框。

## 3. 最后两个选择是否真的刺痛，而不是像普通分支按钮

**这是目前最大的短板，问题不在文字，在呈现。**

- 两个选择的**文案本身**写得不错：玉的两个结局（"失而复得的欢喜"vs"一府的不安"）和女孩的两个结局（"为她写一首判词"vs"这里本该有一首判词，可你没有写"）都有真实的代价，没有"赢"的选项，符合唏嘘基调。
- 但实际打开看，**两个选择按钮长得和游戏里其他任何一个"继续"按钮完全一样**——同样的描边方框、同样的字号、同样的 hover 效果，挨在一起堆叠展示，点哪个都是 0.1 秒的肌肉记忆动作。情绪上刚刚被"井边的女孩"那段文字压住，下一秒却是在点一个跟"前往第二天"视觉上没有区别的 UI 控件——**文字给的重量，被按钮的轻量感卸掉了**。
- 井边那一拍（过桥，"全剧之心"）现在配上了井台遗物那张图，效果很好——这是目前唯一一个画面和情绪同步在加重的地方。**问题是这股重量只撑到看见画面那一刻，一进入抉择二的按钮页就被按钮的轻飘感打回原形。**

**建议方向（不是现在就改，先记录方向）**：
- 两个选择按钮之间需要更大的视觉/心理间距——不只是 `space-y-3` 的物理间距，可以考虑去掉"options as a list"的列表感，让两个选择各自占据更大、更不对称的视觉位置，甚至中间插一句"——你会怎么选？"式的停顿文字。
- 选中后不要立刻跳反应文字，给一个极短的"手悬在那里"的犹豫感（哪怉是按钮按下时的视觉反馈比现在更"重"，比如轻微变暗+延迟）。
- 这两个按钮目前的 hover/active 效果（`hover:border-amber-200/70`）和其他场景的"继续/前往下一幕"按钮共用同一套样式定义，建议从视觉语言上就分裂出"抉择按钮"和"导航按钮"两个不同的家族，玩家一看就知道"这一个不一样"。

## 4. 井边遗物是否成为全剧情绪中心

**是的，目前看是全剧最强的一拍——但它的"强"目前只持续了两拍就过去了，没有被结局回收放大。**

- 画面+文字的同步做得最好的地方就是这里（井台石阶+荷包鞋样+"玉的事，无论你怎么选，到天亮，都会过去。可还有一个人，不会因为天亮，就回来。"）。这是目前唯一一处真正做到"画面先行、情绪自己说话"的地方，其余场次还是靠文字在扛。
- **但终局判词页里，"她的判词"卡片只在选了"remember"才出现**，"leave"时只有一句"这里，本该有一首判词。可你没有写。"——这句留白写得对，但视觉上这张卡和判词卡是同一个 `rounded-md` 灰色方框，**和上面"她的判词"那张卡片几乎一样重**，没有把"被记住"和"被遗忘"两条路径在结局里拉开差距。读完全程后最该被放大的，是这个对比本身。
- **建议方向**：终局里这两种结局（记住/遗忘女孩）应该在版式上做出明显不对称——记住她的那个版本可以让那首判词单独成一屏、配合井边那张图再出现一次（首尾呼应）；忘记她的那个版本可以故意留更多空白、甚至延长停顿时间，让"什么都没有"这件事本身被感觉到，而不是用一句和别处同样大小的灰字一晃而过。

## 5. 终局判词是否让玩家愿意截图或重玩

**判词文字本身质量很高，是这个项目目前写得最讲究的部分；但呈现节奏目前是"和前面所有内容一样的速度"，没有给这张"压轴牌"应有的停顿感。**

- 六首判词（心太软/旁观者/直性子/百变心/偏心眼/高冷系）逐字看过，七言四句+白话引句+判断逻辑的三层结构是这个项目最大的设计亮点，"独有自家无处问""解尽旁人解己难"这类末句确实有"反过来读你"的冲击力，值得截图传播。
- 但终局页是用和 Day1Wrap/Day2Wrap 完全一样的 `DaySummary` 式"点一下出一段"节奏在走的——原句回放、尾声、她的判词、判断逻辑、判词卡片，五段内容用同一个速度依次蹦出来，**判词卡片没有获得比前面四段更长的停留或更强的视觉仪式感**。玩家好不容易走到这里，给的"奖励感"被走流程的速度抹平了。
- **建议方向**：判词卡片（beats[4]）出现时可以单独处理——比如出现前有一个比其他几拍更长的黑屏停顿、卡片本身的淡入比前面几段慢一点、配色更突出（目前已经是 amber 边框，可以再加一点，比如卡片出现时背景图也做一次微妙的强调，如轻微变亮或定格）。这是"判词是给走到最后的人的文学奖励"这句设计初心，目前在节奏上还没有被兑现。

## 总体判断

主链条（开场→Day1三场→Day2两场→Day3两个抉择→终局）**内容上已经成立**，不需要再加人物/支线/系统，这个边界是对的。

**真正要做的不是"加什么"，是把已经写对的几个重点（井边、两个抉择、判词卡）的呈现力度调到匹配它们的叙事分量**——目前内容的情绪曲线已经爬上去了，但呈现节奏是一条平直线，没有跟着内容起伏。这是下一步"剪辑"该做的事，不是"拍摄"（写内容/画图）该做的事。

不建议现在做的事：
- 不要加新人物、新场次、新读法类型
- 不要扩第三天的抉择数量（两个刚好）
- 不要在判词体系上加新原型（六个够了）

建议的下一步顺序：
1. 先把"两个选择按钮视觉太轻"和"判词卡呈现节奏太平"这两处改掉——这两处改完，"刺痛感"和"奖励感"会立刻不一样，性价比最高。
2. 开场"你在谁的身体里"补一句明确的内心确认。
3. Day2Wrap 结尾那句"说出来/咽下去"的措辞对齐一下第三天实际给的两个选择。
4. 终局"记住/遗忘女孩"两条路径在版式上拉开差距。
