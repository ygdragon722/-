# 红楼幻梦

一款以《红楼梦》为背景的竖屏互动短篇。玩家是一位周末独自游览现代仿建大观园的女性读者，因风月鉴入梦，进入贾宝玉的身体，在省亲夜后的三天里读人、追问、做选择，最后由游戏反过来读你一次。

**在线游玩**：https://ygdragon722.github.io/-/

---

## 当前形态

- **三天探案 VN**：开场入梦 → 第一天读人 → 第二天人命 → 第三天双抉择 → 终局判词。
- **全屏水墨演出**：竖屏场景图、人物近景、底部字幕框和逐句点击节奏。
- **读法选择**：每场用不同方式理解人物，不走死胡同，读错也会留下痕迹。
- **道德抉择**：最后围绕“玉”和“无名女孩”做两个选择，没有纯粹胜利路线。
- **反向判词**：结局根据全程读法和最终选择，生成“反过来读你”的判词。

## 项目结构

- `src/story/`：互动短篇主流程、场景组件、结局和存档逻辑
- `src/story/data/`：开场、人物场、过渡、抉择、判词等文本数据
- `public/assets/scenes/`：剧情场景图和过场氛围图
- `public/assets/portraits/`：人物表情近景
- `docs/`：开发计划、审稿记录、美术提示词和体验验收清单
