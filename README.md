# 红楼幻梦

一款以《红楼梦》为背景的桌面网页版恋爱养成游戏。玩家扮演贾宝玉，在大观园中度过三十天，通过日程、探索与对话推进剧情，攻略金陵七钗。

**在线游玩**：https://mashengjia.github.io/红楼幻梦/

---

## 游戏特色

- **满屏 AVG 对白演出**：全屏场景背景 + 角色立绘 + 打字机效果，告别弹窗
- **7 位女主，各 5 段剧情**：初遇、试探、冲突、名场面、结局前夜，共 35 个剧情节点
- **梦册任务簿**：实时追踪每位角色的路线进度与解锁条件
- **MBTI 理解度机制**：选择符合角色性格的对话方式，提升理解度，海棠诗社可借助理解深厚的角色助力
- **章节节奏**：第 15 天海棠诗社考核，第 18/20/22 天危机铺垫，第 25 天抄检大观园生死抉择
- **9 条结局**，深度路线有专属场景文本
